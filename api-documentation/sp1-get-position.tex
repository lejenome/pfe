\textbf{Réponse Codes}

\begin{description}
    \item[\texttt{200 Success}] Dernière position du périphérique spécifié retournée.
    \item[\texttt{404 Not Found}] Aucune position trouvée pour le périphérique spécifié.
\end{description}
